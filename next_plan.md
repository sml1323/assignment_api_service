# Next Plan: 향후 발전 방향

## 아이디어 1: 여행 앨범 협업 서비스
- 주최자가 여행 사진을 페이지당 하나씩 배치하고 글을 적음
- 참여자들은 각 사진 페이지에 텍스트를 추가 ("이때 어땠지" 형태의 회상)
- 각 페이지마다 고정된 수의 영역을 할당, 참여자가 선점하는 방식
- 텍스트 박스 또는 그리기(손글씨) 지원
  - 휴대폰: 손글씨 입력
  - 아이패드: Apple Pencil로 필기/꾸미기
- Canvas 기반 편집기 필요

## 아이디어 2: 축하책 개선
- 표지에 "주최자 외 참여자 N명" 형태로 표시
- 페이지당 1명 분량으로 레이아웃
- 사진이 있을 경우 사진+메시지 배치 규칙 정의
- 여러 가지 축하 템플릿 선택 가능 (심플, 화려, 귀여운 등)
- 미리보기 단계 추가 (주문 전 확인)

## 기술적 고려사항
- Canvas API 또는 Fabric.js로 페이지 편집기
- 영역 선점: WebSocket 또는 폴링으로 실시간 반영
- 손글씨 → 이미지 변환 → Sweetbook API에 업로드
- 페이지 레이아웃 엔진 필요 (사진 크기/위치 자동 조정)
